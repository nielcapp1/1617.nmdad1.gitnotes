Briefing

Een web applicatie die koffiebars uit Gent promoot.

Iedere koffiebar moet een rating krijgen om toegang te krijgen tot een wifi-code, de parkeerlocaties weergeven, de locatie aantonen en hun assortiment aanbieden.

Analyse

Wat bieden we de klant aan?

Een Data-driven Progressive Responsive Web Application waarbij we een open dataset zullen aanspreken van Stad Gent.

Keuze van technologie

HTML5

SASS

Javascript

Open Datasets van stad Gent

Tijdspanne

Het volledige proces moet afgerond zijn in ongeveer 10 weken. De deadline valt op 23 december 2016.
